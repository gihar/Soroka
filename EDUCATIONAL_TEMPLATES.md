# Образовательные шаблоны протоколов

## Обзор

Система протоколирования теперь включает специализированные шаблоны для образовательных мероприятий. Эти шаблоны оптимизированы для документирования различных форматов обучения и обеспечивают структурированное сохранение учебного контента.

## Категория: Educational

Добавлена новая категория шаблонов `educational` - "Образовательные встречи".

## Доступные шаблоны

### 1. Лекция и презентация (`education_lecture`)
**Назначение:** Для академических лекций, презентаций и теоретических занятий

**Ключевые секции:**
- Цели обучения
- Структура лекции
- Ключевые концепции и определения
- Основной материал
- Примеры и кейсы
- Вопросы и ответы
- Домашнее задание
- Дополнительные материалы

**Переменные:** `learning_objectives`, `key_concepts`, `examples_and_cases`, `questions_and_answers`, `homework`, `additional_materials`

### 2. Тренинг и практикум (`education_training`)
**Назначение:** Для практических тренингов с акцентом на отработку навыков

**Ключевые секции:**
- Цели обучения
- Теоретическая база
- Практические упражнения
- Разбор кейсов
- Работа в группах
- Сессия обратной связи
- Ресурсы и материалы

**Переменные:** `practical_exercises`, `examples_and_cases`, `group_work`, `feedback_session`, `materials`

### 3. Семинар и дискуссия (`education_seminar`)
**Назначение:** Для интерактивных семинаров и дискуссионных занятий

**Ключевые секции:**
- Цели семинара
- План дискуссии
- Анализ дискуссии
- Вклад участников
- Ключевые точки мнений
- Практические задания
- Материалы для самостоятельной работы

**Переменные:** `dialogue_analysis`, `participant_contributions`, `controversial_points`, `questions_and_answers`

### 4. Мастер-класс (`education_masterclass`)
**Назначение:** Для демонстрации экспертных техник и профессиональных секретов

**Ключевые секции:**
- Экспертность мастера
- Демонстрация техник
- Профессиональные секреты
- Реальные кейсы
- Практика под руководством
- Рекомендуемые инструменты

**Переменные:** `practical_demonstration`, `professional_secrets`, `examples_and_cases`, `audience_practice`

### 5. Воркшоп и работа в группах (`education_workshop`)
**Назначение:** Для интенсивной групповой работы и коллаборации

**Ключевые секции:**
- Формирование групп
- Групповые упражнения
- Результаты работы групп
- Взаимная обратная связь
- Индивидуальные рефлексии
- Методические материалы

**Переменные:** `group_formation`, `practical_exercises`, `group_results`, `peer_feedback`, `individual_reflections`

### 6. Вебинар и онлайн-лекция (`education_webinar`)
**Назначение:** Для онлайн-форматов обучения и вебинаров

**Ключевые секции:**
- Программа вебинара
- Результаты опросов
- Вопросы из чата
- Демонстрация в реальном времени
- Материалы для скачивания
- Практические задания

**Переменные:** `platform`, `poll_results`, `chat_questions`, `live_demonstration`, `downloadable_materials`

## Специализированные переменные

### Основные образовательные переменные:
- `learning_objectives` - цели обучения (что должны узнать/научиться делать)
- `key_concepts` - основные понятия и определения
- `practical_exercises` - практические задания и упражнения
- `materials` - учебные материалы и ресурсы

### Специализированные переменные:
- `examples_and_cases` - примеры и реальные кейсы
- `homework` - домашние задания
- `additional_materials` - дополнительные материалы для изучения
- `dialogue_analysis` - анализ дискуссий
- `participant_contributions` - вклад участников
- `professional_secrets` - профессиональные секреты и техники

## Интеграция с ML-системой

### Автоматическое распознавание
Система автоматически определяет образовательные встречи по ключевым словам:
- лекция, презентация, тренинг, семинар
- мастер-класс, практикум, воркшоп, вебинар
- обучение, учебный, образовательный, теория, практика

### Умные рекомендации
ML-модель анализирует транскрипцию и рекомендует наиболее подходящий образовательный шаблон на основе:
- типа образовательного мероприятия
- содержания и структуры занятия
- интерактивных элементов

### Специализированные промпты
Для образовательных встреч используются специализированные промпты, которые:
- структурируют учебный материал
- выделяют ключевые концепции
- фиксируют примеры и практические задания
- определяют цели обучения

## Использование

1. **Автоматический выбор:** Система автоматически распознает тип образовательной встречи и предлагает подходящий шаблон

2. **Ручной выбор:** Пользователь может выбрать любой из 6 образовательных шаблонов в зависимости от формата мероприятия

3. **Гибкая настройка:** Шаблоны поддерживают все стандартные переменные плюс специализированные образовательные поля

## Пример использования

```python
# Получение образовательных шаблонов
from src.services.template_library import TemplateLibrary

library = TemplateLibrary()
edu_templates = library.get_educational_templates()

# Выбор шаблона для лекции
lecture_template = next(t for t in edu_templates if t['id'] == 'education_lecture')

# Данные для рендеринга
lecture_data = {
    "date": "20 ноября 2024",

    "materials": "Учебник, ноутбук с Python, TensorFlow"
}
```

## Преимущества

- **Структурированность:** Четкая организация учебного материала
- **Педагогическая точность:** Специализированная обработка образовательного контента
- **Гибкость:** Поддержка различных форматов обучения
- **Интеграция:** Бесшовная работа с существующей системой
- **ML-оптимизация:** Умный выбор шаблона на основе анализа контента
