# Руководство пользователя

## Начало работы

1.  Запустите бота в Telegram: `/start`
2.  Бот поприветствует вас и будет готов принимать файлы.

## Обработка встреч

### Вариант 1: Загрузка файла
Просто перетащите аудио или видео файл в чат.
-   **Поддерживаемые форматы**: MP3, WAV, M4A, OGG, MP4, MOV и др.
-   **Лимит**: 20MB (ограничение Telegram).

### Вариант 2: Внешние ссылки
Отправьте прямую ссылку на файл из:
-   **Google Drive** (должен быть публично доступен или расшарен для бота, если настроено)
-   **Яндекс.Диск**
-   **Прямая ссылка** (например, `https://example.com/meeting.mp3`)

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Инициализация бота |
| `/help` | Показать справку |
| `/settings` | Настройка LLM провайдера и других параметров |
| `/templates` | Управление шаблонами встреч |
| `/performance` | Показать статистику производительности (Только для админов) |
| `/health` | Проверка здоровья системы (Только для админов) |

## Шаблоны встреч

Soroka поддерживает различные шаблоны для структурирования протоколов:
-   **Общий**: Стандартная структура встречи.
-   **Daily Standup**: Для agile команд.
-   **Стратегическая сессия**: Планирование высокого уровня.
-   **Технический обзор**: Для инженерных обсуждений.

Вы можете выбрать шаблон после загрузки файла или позволить боту **автоматически определить** лучший шаблон на основе контента.

## Обратная связь и Улучшения

После получения протокола вы можете:
-   **Перегенерировать**: Попробовать снова с другой LLM или шаблоном.
-   **Редактировать**: Запросить изменения конкретных секций.
